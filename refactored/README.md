# How to set up Repo for Development:

##### installation
1. use wsl2
2. >> sudo apt update
3. >> sudo apt upgrade
4. >> sudo apt install npm
5. >> npm install --global yarn

##### setup backend
1. >> cd backend
2. >> source venv/bin/activate
3. >> pip3 install Flask

##### setup frontend
1. >> cd frontend
2. >> yarn install



# How to run the application:

##### start backend
1. >> cd backend
2. >> source venv/bin/activate
3. >> python3 backend.py

##### start frontend
1. >> cd frontend
2. >> yarn start
